# Exoplanet Habitability Prediction System

## 📌 Описание

Данная система предназначена для оценки вероятности потенциальной обитаемости экзопланет на основе их физических характеристик с использованием методов машинного обучения.

Модель обучена на наборе данных экзопланет и использует ключевые параметры, влияющие на условия существования жизни.

---

## 📥 Ввод данных

Пользователю необходимо ввести следующие параметры:

- **Радиус планеты** (в радиусах Земли)  
- **Расстояние до звезды** (в астрономических единицах, AU)  
- **Внешняя граница обитаемой зоны** (в AU)  
- **Температура планеты** (в Кельвинах, пример: Земля ≈ 288 K)  

---

## 📤 Результат

Система выводит:

> вероятность потенциальной обитаемости планеты (от 0 до 1)

---

## 📊 Особенности модели

- Используется алгоритм **Random Forest**
- Учитывается дисбаланс классов (`class_weight='balanced'`)
- Основной значимый параметр - температура планеты
- Применяется логарифмирование признаков

---

## ⚠️ Ограничения

- Данные имеют сильный дисбаланс классов
- Модель обучена на наблюдаемых экзопланетах, а не на реально обитаемых мирах
- Результат является вероятностной оценкой, а не точным предсказанием
